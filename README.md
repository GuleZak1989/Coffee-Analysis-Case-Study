# Coffee-Analysis-Case-Study
The Coffee Analysis file provides a detailed breakdown of sales data from January 1 to June 30, 2023, across three New York locations: Astoria, Hell’s Kitchen, and Lower Manhattan. 

# Summary of Findings
•	Slowest periods are Evenings, followed by Afternoons, making them ideal for targeted marketing campaigns.
•	Products like Cappuccino Lg, Latte Rg, and Brazilian Lg dominate sales. These should be prioritized for stock.
•	Items such as Hazelnut Biscotti and certain flavored syrups are underperforming—ideal candidates for promotional campaigns.
